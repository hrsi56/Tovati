# דו״ח מסירה — טובתי 1.6.0

מנוע: `bbt-fusion-2.3.0`  
גיבוי: schema v5; שחזור v1–v5  

## מה נוסף

- ב־onboarding נוספו השאלות „מה אורך המחזור שלך בדרך כלל?” ו„מה אורך הווסת הממוצע?”.
- אורך המחזור שמסרה המשתמשת משמש prior התחלתי בלבד: אחרי מחזור מלא אחד ושניים משקלו דועך, ומשלושה מחזורים מלאים ההיסטוריה בפועל מחליפה אותו.
- אורך הווסת שמסרה המשתמשת מסמן ביומן את יום הסיום המשוער בתחילת השימוש, עם סימון מפורש במקרא.
- תחת „אודות” נוסף קישור ל־LinkedIn של מפתח האפליקציה.
- בסוף מסך ההגדרות נוסף „בואי נגדיר מחדש”. הוא פותח את ה־onboarding עם הערכים הקיימים ומשנה הגדרות בלבד; מחזורים, מדידות ותצפיות אינם נמחקים או נכתבים מחדש.
- גיבוי schema v5 כולל את שתי תשובות ה־onboarding, כדי שישוחזרו במכשיר חדש.

## בהירות ומגבלות

- הערכת ה־onboarding מוצגת כהערכת פתיחה ולא כעובדה או כאישור ביוץ.
- גם כאשר קיימת הערכת פתיחה, מהימנות המנוע אינה עולה אוטומטית מעבר ל„לא מספקת” בלי היסטוריה או סימנים ביולוגיים.
- LH, נוזל וחום ממשיכים להיות אותות ספציפיים יותר ויכולים להזיז את אומדן לוח־השנה.
- תחזית עתידית נשארת צמד ימים; הערכת ביוץ בדיעבד נשארת יום יחיד.
